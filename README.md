### Usage
command-line

    chmod u+x ./pangaea.pl 

    ./pangaea.pl 

    ./pangaea.pl n0 j6 

    ./pangaea.pl : j6 

    ./pangaea.pl ? j6 

    ./pangaea.pl cgdae n0 j3 j36 

    ./pangaea.pl cgdae: j3 j36 j236 

    ./pangaea.pl cgdae? '^([jk]\d)+$' 

    ./pangaea.pl cgdae gamut | less
